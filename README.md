# Music Player Equalizer

This project is a music player equalizer that analyzes audio signals, identifies noise components, and designs filters to improve the audio quality.

## Project Structure

*   `audio/`: Contains the audio files in mp3 and wav format.
*   `section1/`: Contains the MATLAB script and answers for Section 1: Signal Acquisition.
*   `section2/`: Contains the MATLAB script and answers for Section 2: Frequency Domain Analysis.
*   `section3/`: Contains the MATLAB script and answers for Section 3: Filter Design and Implementation.
*   `section4/`: Contains the MATLAB script and answers for Section 4: Performance Evaluation.
*   `website/`: Contains a simple website to display the results.

## How to Run

1.  Open the MATLAB scripts in each section folder.
2.  Run the scripts to see the plots and results.
3.  Open `website/index.html` in a web browser to see the summary of the project.
